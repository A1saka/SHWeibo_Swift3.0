# 纯Swift 3高仿仿微博应用
最近学习 Swift 3， 自己动手写了个高仿微博应用，权当练手

* 目前只弄完主页功能，后续如果有时间可能会添加发微博，评论等功能
* 如果后面有时间的话会画出整个项目流程控制图 （坑挖得有点多 - -! ）
* 项目只用于 Swift 学习交流，如果有什么问题请 issues，如果可以的话顺手点个 Star，非常感谢 ：）

---

# 使用
* 项目使用 CocoaPods 管理，运行前请 `pod install`
